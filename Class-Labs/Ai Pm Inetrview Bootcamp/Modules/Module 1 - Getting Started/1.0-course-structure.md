# Lesson 1.0: Course Structure

![banner](./images/banner.png)

Welcome to the AI PM Interview Bootcamp! This lesson will help you understand the course structure, learning path, and what to expect throughout your journey.

---

## Course Overview

This bootcamp is designed to help Product Managers master Claude Code and leverage AI tools to enhance their product management workflows. The course is structured into **three progressive modules** that build upon each other, providing hands-on experience with real Product Management tasks.

---

## Course Structure

The bootcamp consists of three main modules:

### Module 1: Getting Started
**Foundation and Setup**

This module introduces you to Claude Code and helps you set up your development environment. You'll learn:
- What Claude Code is and its capabilities
- Why Claude Code matters for Product Managers
- How to install and configure Claude Code
- Setting up Claude Code in Visual Studio Code

**Lessons:** 5 lessons (1.0 - 1.4)

---

### Module 2: Understanding Claude Code Context System
**Mastering Context for PM Tasks**

This module focuses on mastering Claude Code's context system and using it for essential Product Management tasks. You'll learn:
- Organizing files for optimal context management
- Understanding and using Claude's context system
- Performing market research with context
- Conducting user research with context
- Writing professional PRDs using context

**Lessons:** 6 lessons (2.0 - 2.5)

---

### Module 3: Building Agents & Visualizations
**Advanced AI-Powered Tools**

This module teaches you to build advanced AI-powered tools and visualizations. You'll learn:
- Creating autonomous AI agents
- Building interactive websites and mockups
- Generating data visualizations and dashboards
- Transforming data into actionable insights

**Lessons:** 3 lessons (3.0 - 3.2)

## How to Navigate This Course

1. **Start with Module 1** - Begin with Lesson 1.0 (this lesson) and work through sequentially
