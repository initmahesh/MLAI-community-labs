# Module 1: Getting Started

![images](./images/banner.png)

Welcome to Module 1! This module will guide you through the fundamentals of Claude Code and help you set up your development environment to start using this powerful AI assistant.

---

## Module Overview

In this module, you will learn how to:
- Understand what Claude Code is and its capabilities
- Discover why Claude Code is valuable for Product Managers
- Install and configure Claude Code on your local machine
- Set up Claude Code in Visual Studio Code

---

## Learning Objectives

By the end of this module, you will be able to:
- ✅ Understand Claude Code's features and capabilities
- ✅ Recognize how Claude Code can enhance Product Management workflows
- ✅ Install Node.js and verify your system requirements
- ✅ Install and configure Claude Code locally
- ✅ Set up Claude Code in Visual Studio Code
- ✅ Use Claude Code for basic tasks in your development environment

---

## Prerequisites

Before starting this module, make sure you have:

1. **A Claude Subscription** - A paid Claude plan (Pro or Max) is required
2. **A Computer** - Windows, macOS, or Linux
3. **Internet Connection** - Required for installation and usage

---

## Quick Start Guide

1. **Start with Lesson 1.0** - Understand the course structure
2. **Continue to Lesson 1.1** - Learn what Claude Code is
3. **Read Lesson 1.2** - Discover why it matters for Product Managers
4. **Complete Lesson 1.3** - Install Claude Code on your machine
5. **Finish with Lesson 1.4** - Set up Claude Code in VS Code

Each lesson builds on the previous one, so we recommend following them in order.
